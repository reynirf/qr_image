# QRImage

<p align="center">
    QRImage is a web app to generate QR codes with images.
</p>

<p align="center">
    <img src="images/demo.png">
</p>


### Usage

1. Open index.html
2. Upload an image.
3. Enter the URL.
4. Adjust parameters.
5. Download `PNG`.

## Dependency

* A [fork](https://github.com/reynirf/qrcodejs) of [davidshimjs / qrcode](https://github.com/davidshimjs/qrcodejs)
